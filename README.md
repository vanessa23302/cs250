1. Summarize the project and what problem it was solving.
For this project, I modified an existing slideshow application to reflect new requirements from the Product Owner. The original version featured general travel destinations, but the updated request shifted the focus to wellness and detox travel. My goal was to revise the destination list, images, and descriptions to better align with this new theme and help the SNHU Travel project stay relevant in the current travel market.

2. What did you do particularly well?
I think I did a good job updating the content without breaking the original structure of the program. I was able to correctly replace the images, descriptions, and links for the new destinations while keeping the overall layout and functionality intact. I also made sure to include clear comments to explain my changes and make the code easier to understand for others.

3. Where could you enhance your code? How would these improvements make your code more efficient, secure, etc?
If I were to improve this code, I would find a way to pull the destination information from an external file instead of hardcoding it. That way, it would be easier to update or expand in the future without having to touch the source code. This would make the app more efficient and maintainable, especially as the list of destinations grows or changes again.

4. Did you find writing any piece of this code challenging and how did you overcome this? What tools and/or resources are you adding to your support network?
One part I struggled with was getting the images to load properly using the getResource() method. At first, I kept running into issues where the app couldn’t find the files. I fixed this by double-checking the file paths and making sure the images were in the correct resources folder. To get through it, I also looked at a few Stack Overflow threads and used Eclipse to step through the code and troubleshoot what was going wrong.

5. What skills from this project will be particularly transferable to other projects and/or coursework?
This project really helped me practice updating existing code and working within a specific set of requirements, which I know will be important in future projects. I also got more comfortable using Java Swing for GUI development and managing image resources, both of which will definitely come in handy in other classes and eventually in a development job.
